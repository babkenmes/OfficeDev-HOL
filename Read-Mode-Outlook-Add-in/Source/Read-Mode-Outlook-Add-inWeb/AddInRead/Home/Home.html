HOL: Outlook Add-in (Read Mode)

This sample demonstrates a few different ways to interact with the Office context. Accessing different types of data for a mailbox item (message or appointment) in read mode.

Exercise: Get subject

Get the item subject and display it by clicking the button down below.

Exercise: Get sender

Get the item sender and display it by clicking the button down below.

Exercise: Get body

Get the item body by clicking the button down below. This requires a minimum mailbox requirement set version of 1.3.
