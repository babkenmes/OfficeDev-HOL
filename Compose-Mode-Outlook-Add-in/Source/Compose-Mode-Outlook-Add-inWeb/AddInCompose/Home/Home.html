HOL: Outlook Add-in
(Compose Mode)

This sample demonstrates a few different ways to interact with the Office context. Setting different properties of a mailbox item (message or appointment) in compose mode.

Exercise: Set subject

Set the item subject by clicking the button down below.

Exercise: Set recipients

Set the item recipients by clicking the button down below.

Exercise: Set body

Set the item body by clicking the button down below. This requires a minimum mailbox requirement set version of 1.3.

Office UI Fabric
